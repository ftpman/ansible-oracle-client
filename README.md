# Ansible: oracle-client

Script to setup Oracle client/library files

Available on Ansible Galaxy: [pgkehle.oracle-client](https://galaxy.ansible.com/pgkehle/oracle-client)


# Examples

```YAML

  - hosts: all
  
    roles:
      - pgkehle.oracle-client
```
